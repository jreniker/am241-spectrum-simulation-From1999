# am241-spectrum-simulation-From1999

The original code for this was writting in 1999 as part of my Masters thesis.  I started to write this in C and XLib 
but my advisor did not understand C (not that I was an expert...) so I switched to QuickBasic which he easily 
understood.  After 26 years, as a fun project, I wanted to resurrect this and used v0.app.

This is a [Next.js](https://nextjs.org) project bootstrapped with [v0](https://v0.app).

## Built with v0 after my initial published release (my thesis could be found by searching Google books once upon a time).
